# Tax Calculator

This is a simple tax calculator that estimates the tax based on the income and the tax brackets.

> **DISCLAIMER**: This is a simple tax calculator and should not be used for any real-world tax 
> calculations. I am not a tax or finance professional and am not above mistakes. This tool 
> likely overlooks certain real-world tax complexities. Always consult a professional.

## Development Quickstart

Make sure you have Rust and Cargo installed. Then run:

```
cargo run
```
